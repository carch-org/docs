# 🖥️ CLI 命令

Carch 也支持 CLI 命令！使用 `carch` 配合以下选项：

:::code-group

```sh [⚙️ CLI]
carch --help
```

:::

## 🔧 命令详解

| 🏷️ 命令            | 📄 描述                                                                                      |
|--------------------|----------------------------------------------------------------------------------------------|
| `--help`, `-h`     | 显示此帮助信息并退出。                                                                       |
| `--version`, `-v`  | 显示程序版本。                                                                               |
| `--log`            | 仅为当前会话启用日志记录，位于 `~/.config/carch/carch.log`。                                 |
| `--no-cleanup`     | 禁用 `~/.cache/carch` 目录中模块脚本的缓存清理。                                            |
| `check-update`     | 检查是否有新版本可用。                                                                       |
| `update`           | 交互式更新 Carch。                                                                          |
| `uninstall`        | 交互式卸载 Carch。
