# ⌨ TUI 键位绑定

在 Carch 中，我们有一些用于导航和与 TUI 交互的键位绑定。您始终可以通过在 Carch TUI 中按 `?` 来查看所有绑定。

以下是一些关于键位绑定的清晰信息，可能帮助您理解如何正确使用它们。

---

## 导航

我们使用 vim 导航键进行移动（上、下、左、右）。

- `j` → 向下移动
- `k` → 向上移动
- `h` → 向左移动
- `l` → 向右移动

您也可以使用箭头键进行移动：

- `↑` → 向上移动
- `↓` → 向下移动
- `←` → 向左移动
- `→` → 向右移动

您可以偏好使用 vim 键或箭头键，选择您觉得舒服的方式。

---

## 运行脚本

对于快速操作，如运行脚本：

- 导航到右侧的任何脚本，然后按 `Enter` 运行它。
- 如果您是 Neovim 爱好者，也可以按 `l` 来运行它。

当确认提示出现时：

- 按 `y`、`l` 或 `→` 确认并运行。
- 按 `n`、`h`、`Esc`、`q` 或 `←` 取消。

---

## 多选模式

您不能同时运行多个脚本，但您可以选择多个脚本，它们将按照您选择的顺序一个接一个地运行。

要使用多选功能：

- 按 `m` 进入多选模式
- 按 `空格键` 选择脚本
- 按 `Enter` 运行它们
- 按 `Esc` 或 `q` 退出多选模式

它们将逐个执行，每个脚本完成后才会运行下一个。

---

## 主题

Carch TUI 也有多个主题。您可以设置一个在会话间记住的默认主题。

- 按 `t` 打开主题选择器
- 使用 `j`/`k` 或 `↑`/`↓` 浏览主题
- 按 `Enter` 将所选主题设置为默认
- 按 `Esc`、`q` 或 `t` 关闭而不更改

所选主题已保存，将在未来启动时使用。您也可以使用 `--set-theme` CLI 标志从终端设置。

---

## 搜索

我们还有一个搜索选项，方便您快速找到任何脚本，而无需来回导航。

- 按 `/` 进入搜索模式
- 输入脚本名称
- 您可以使用 Tab 键自动补全
- 按 `Enter` 导航到该脚本

---

## 预览

如果您想在运行前检查脚本：

- 按 `p` 预览脚本

这有助于您在执行脚本之前了解它的功能。

---

## 描述

描述模式显示有关脚本的简短信息。这对初学者尤其有帮助，可以了解脚本的内容。

- 转到右侧的脚本列表
- 按 `d` 查看所选脚本的描述
- 按 `Esc`、`q` 或 `d` 关闭

---

## 帮助

- 随时按 `?` 打开显示所有按键绑定的帮助弹窗
- 按 `Esc`、`q` 或 `?` 关闭

## 快速导航

- 按 `Home` 跳转到当前列表顶部
- 按 `End` 跳转到底部
