# Скрипты

<img src="https://cdn-icons-png.flaticon.com/128/3721/3721643.png" width="40" />

**Carch поставляется со следующими скриптами для настройки различных аспектов ваших дистрибутивов на базе Arch, Fedora и openSUSE.**

### Описания:

### Dwm
- *dwm - это динамический тайловый оконный менеджер для X11, известный своей минималистичностью, быстротой и настраиваемостью через редактирование исходного кода. Скрипт применит мою предварительно настроенную установку dwm из моего репозитория [harilvfs/dwm](https://github.com/harilvfs/dwm) и установит все необходимые зависимости, чтобы сделать среду полностью функциональной.*

### Hyprland
- *Hyprland - это динамический тайловый композитор Wayland, поддерживающий современные функции, такие как анимация, оконные эффекты и гибкие тайловые макеты. Скрипт позволяет устанавливать предварительно настроенные установки Hyprland от уважаемых пользователей - [ml4w](https://github.com/mylinuxforwork/dotfiles), [end-4](https://github.com/end-4/dots-hyprland), [Jakoolit](https://github.com/JaKooLit/Arch-Hyprland) и [prasanthrangan dots](https://github.com/prasanthrangan/hyprdots).*

### Шрифты
- *Загружает и устанавливает различные шрифты Nerd для улучшения читаемости и эстетики в терминальных приложениях.*

### LTS Ядро
- *Этот скрипт устанавливает LTS ядро только для дистрибутивов на базе Arch (не для Fedora или openSUSE).*

### Пакеты
- *Предоставляет меню с различными категориями пакетов, такими как инструменты Android, браузеры, мультимедиа, разработка и другие. Включены наиболее широко используемые пакеты, дополнительные пакеты будут добавлены в будущих обновлениях.*

### Alacritty
- *Alacritty - это быстрый эмулятор терминала с GPU-ускорением, сфокусированный на производительности и простоте. Скрипт применяет мою предварительно настроенную конфигурацию Alacritty к вашей системе.*

### AUR
- *AUR (Arch User Repository) - это управляемый сообществом репозиторий для пользователей Arch Linux, содержащий пользовательские скрипты сборки пакетов (PKGBUILDs), которые облегчают установку программного обеспечения, недоступного в официальных репозиториях. Скрипт позволит вам выбрать помощника AUR и установить его согласно вашим предпочтениям.*

### Fastfetch
- *Fastfetch - это легкий инструмент системной информации, отображающий ОС, аппаратное обеспечение и другие детали в чистом, настраиваемом формате. Скрипт применяет мою конфигурацию fastfetch из репозитория [harilvfs/fastfetch](https://github.com/harilvfs/fastfetch).*

### GRUB
- *GRUB (GNU GRand Unified Bootloader) - это загрузчик, который позволяет выбирать и загружать операционные системы или ядра при запуске. Скрипт предлагает различные темы и применяет необходимые конфигурации GRUB. Благодарность [chrisTitusTech/Top-5-Bootloader-Themes](https://github.com/chrisTitusTech/Top-5-Bootloader-Themes) за темы и помощь со скриптом.*

### Kitty
- *Kitty - это быстрый эмулятор терминала с GPU-ускорением и современными функциями, такими как лигатуры, вкладки, тайловые макеты и поддержка изображений. Скрипт применяет мою предварительно настроенную конфигурацию Kitty.*

### Neovim
- *Neovim - это современный, улучшенный форк Vim, предназначенный для большей расширяемости, скорости и простоты интеграции с плагинами. Скрипт позволяет выбирать между установками Kickstart Neovim и NvChad для настройки вашей IDE-среды.*

### Picom
- *Picom - это легкий композитор для X11, добавляющий прозрачность окон, тени и анимации. Скрипт предлагает установку моей предварительно настроенной конфигурации Picom и поддержку анимированного Picom из [picom-ftlabs](https://github.com/r0-zero/picom).*

### Rofi
- *Rofi - это легкий лаунчер приложений, переключатель окон и замена dmenu для X11. Скрипт применяет мою предварительно настроенную конфигурацию Rofi.*

### SDDM
- *SDDM (Simple Desktop Display Manager) - это современный графический менеджер входа на базе Qt. Скрипт устанавливает тему на базе Catppuccin для SDDM от сообщества Catppuccin.*

### Темы и Иконки
- *Устанавливает и применяет коллекцию тем и иконок для улучшения внешнего вида рабочего стола.*

### Обои
- *Загружает коллекцию красивых обоев Nord и Catppuccin.*

### Tmux
- *tmux - это терминальный мультиплексор, позволяющий управлять несколькими терминальными сессиями в одном окне. Скрипт применяет мою предварительно настроенную конфигурацию tmux.*

### Helix
- *Helix - это модальный текстовый редактор, вдохновленный Vim, сфокусированный на скорости и современных функциях, таких как подсветка синтаксиса, интеграция с языковым серверным протоколом (LSP) и множественные курсоры. Скрипт применяет мою предварительно настроенную конфигурацию Helix.*

### Bash Prompt
- *Настраиваемая командная строка для оболочки Bash. Скрипт позволяет выбирать между темами Tokyo Night, Catppuccin и Nord.*

### Zsh Prompt
- *Zsh (Z shell) - это продвинутая Unix-оболочка с функциями улучшенного автодополнения по табуляции, лучшими скриптами и мощной настройкой. Скрипт устанавливает все необходимые зависимости и применяет мою пользовательскую тему.*

### Fish
- *Fish (Friendly Interactive Shell) - это удобная для пользователя Unix-оболочка с умными автопредложениями, подсветкой синтаксиса и простыми скриптами. Скрипт устанавливает и применяет тему [tide](https://github.com/IlanCosman/tide).*

### Bun
- *Bun - это быстрая многофункциональная среда выполнения JavaScript, сборщик и менеджер пакетов, сфокусированный на производительности. Скрипт устанавливает Bun в вашу систему.*

### Npm
- *npm (Node Package Manager) - это стандартный менеджер пакетов для Node.js, используемый для управления JavaScript-библиотеками и зависимостями. Скрипт устанавливает npm в вашу систему.*

### Chaotic AUR
- *Chaotic-AUR - это популярный неофициальный репозиторий-помощник Arch User Repository (AUR), предоставляющий предварительно собранные бинарные пакеты. Скрипт устанавливает Chaotic-AUR и настраивает `pacman.conf` для его использования.*

### Dunst
- *Dunst - это легкий, настраиваемый демон уведомлений для Linux, который показывает минимальные, ненавязчивые уведомления рабочего стола. Скрипт применяет мою предварительно настроенную установку Dunst.*

### Foot
- *Foot - это легкий, быстрый, минималистичный эмулятор терминала для Wayland, разработанный для простоты и производительности. Скрипт применяет мою предварительно настроенную конфигурацию Foot.*

### Ghostty
- *Ghostty - это быстрый, многофункциональный, кроссплатформенный эмулятор терминала, использующий нативный UI и GPU-ускорение. Скрипт применяет мою предварительно настроенную конфигурацию Ghostty.*

### Аудио
- *Устанавливает и настраивает аудиосистему PipeWire.*

### Bluetooth
- *Устанавливает пакеты Bluetooth и настраивает функциональность Bluetooth.*

### Яркость
- *Устанавливает яркость экрана согласно вашим предпочтениям, используя Brightnessctl.*

### Расширения
- *Рекомендует полезные расширения для браузеров на базе Chromium или Firefox.*

### Bspwm
- *Bspwm — это минималистичный настраиваемый тайлинговый менеджер окон для X11, который использует бинарное разбиение пространства и полагается на внешний демон горячих клавиш для ввода с клавиатуры. Этот скрипт устанавливает конфигурацию Bspwm из [gh0stzk/dotfiles](https://github.com/gh0stzk/dotfiles). Этот установочный скрипт предназначен только для Arch, для других дистрибутивов смотрите [репозиторий](https://github.com/gh0stzk/dotfiles).*
