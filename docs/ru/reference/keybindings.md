# ⌨ Привязки клавиш TUI

В Carch есть несколько привязок клавиш, которые используются для навигации и взаимодействия с TUI. Вы всегда можете увидеть все привязки, нажав `?` внутри TUI Carch.

Вот некоторая понятная информация о привязках, которая может помочь вам понять, как правильно их использовать.

---

## Навигация

Мы используем клавиши навигации vim для перемещения (вверх, вниз, влево, вправо).

- `j` → переместиться вниз
- `k` → переместиться вверх
- `h` → переместиться влево
- `l` → переместиться вправо

Вы также можете использовать клавиши со стрелками для перемещения:

- `↑` → переместиться вверх
- `↓` → переместиться вниз
- `←` → переместиться влево
- `→` → переместиться вправо

Вы можете предпочесть клавиши vim или клавиши со стрелками, в зависимости от того, что вам удобнее.

---

## Запуск скриптов

Для быстрых действий, таких как запуск скрипта:

- Перейдите к любому скрипту в правой части и нажмите `Enter`, чтобы запустить его.
- Если вы энтузиаст Neovim, вы также можете нажать `l`, чтобы запустить его.

При появлении запроса подтверждения:

- Нажмите `y`, `l` или `→` для подтверждения и запуска.
- Нажмите `n`, `h`, `Esc`, `q` или `←` для отмены.

---

## Режим множественного выбора

Вы не можете запускать несколько скриптов одновременно, но вы можете выбрать несколько скриптов, и они будут запускаться один за другим в выбранном вами порядке.

Чтобы использовать множественный выбор:

- Нажмите `m`, чтобы войти в режим множественного выбора
- Нажмите `Пробел`, чтобы выбрать скрипты
- Нажмите `Enter`, чтобы запустить их
- Нажмите `Esc` или `q` для выхода из режима множественного выбора

Они будут выполняться один за другим после завершения каждого.

---

## Темы

Carch TUI также имеет несколько тем. Вы можете установить тему по умолчанию, которая будет запомнена между сеансами.

- Нажмите `t`, чтобы открыть окно выбора темы
- Используйте `j`/`k` или `↑`/`↓` для навигации по темам
- Нажмите `Enter`, чтобы установить выбранную тему по умолчанию
- Нажмите `Esc`, `q` или `t`, чтобы закрыть без изменений

Выбранная тема сохраняется и будет использоваться при будущих запусках. Вы также можете использовать флаг `--set-theme` в CLI, чтобы установить ее из терминала.

---

## Поиск

У нас также есть опция поиска на случай, если вы захотите быстро найти какой-либо скрипт, не перемещаясь взад-вперед.

- Нажмите `/`, чтобы войти в режим поиска
- Введите имя скрипта
- Вы можете использовать автозаполнение с помощью клавиши Tab
- Нажмите `Enter`, чтобы перейти к этому скрипту

---

## Предпросмотр

Если вы хотите проверить скрипт перед его запуском:

- Нажмите `p`, чтобы просмотреть скрипт

Это поможет вам понять, что делает скрипт, прежде чем выполнять его.

---

## Описание

Режим описания показывает краткую информацию о скрипте. Это особенно полезно для новичков, чтобы понять, о чем скрипт.

- Перейдите в правую часть, где находится список скриптов
- Нажмите `d`, чтобы увидеть описание выбранного скрипта
- Нажмите `Esc`, `q` или `d` для закрытия

---

## Справка

- Нажмите `?` в любое время, чтобы открыть окно справки со всеми сочетаниями клавиш
- Нажмите `Esc`, `q` или `?` для закрытия

## Быстрая навигация

- Нажмите `Home`, чтобы перейти в начало текущего списка
- Нажмите `End`, чтобы перейти в конец
