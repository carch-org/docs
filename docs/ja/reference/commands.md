# 🖥️ CLIコマンド

CarchはCLIコマンドもサポートしています！以下のオプションを指定して`carch`を使用してください：

:::code-group

```sh [⚙️ CLI]
carch --help
```

:::

## 📦 使用方法

carch [オプション] [コマンド]

---

## 🔧 コマンドの説明

| 🏷️ コマンド                | 📄 説明                                                                                          |
|----------------------------|---------------------------------------------------------------------------------------------------|
| `check-update`             | 新しいバージョンが利用可能か確認します。                                                              |
| `completions <SHELL>`      | bash、zsh、fish用のシェル補完を生成します。                                                |
| `update`                   | 対話形式でCarchをアップデートします。                                                                       |
| `uninstall`                | 対話形式でCarchをアンインストールします。                                                                    |
| `help`                     | このメッセージまたは指定されたサブコマンドのヘルプを表示します。                                           |
| `--version`, `-v`          | プログラムのバージョンを表示します。                                                                         |
| `--set-theme <THEME>`      | デフォルトテーマを設定して保存します（catppuccin-mocha、dracula、gruvbox、nord、rose-pine、またはdefaultでリセット）  |
| `--help`, `-h`             | このヘルプメッセージを表示して終了します。                                                                  |
