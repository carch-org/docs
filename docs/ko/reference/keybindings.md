# ⌨ TUI 키바인딩

Carch에는 TUI를 탐색하고 상호작용하는 데 사용되는 몇 가지 키바인딩이 있습니다. Carch TUI 내에서 `?`를 누르면 언제든지 모든 키바인딩을 확인할 수 있습니다.

다음은 키바인딩을 올바르게 사용하는 방법을 이해하는 데 도움이 될 수 있는 명확한 정보입니다.

---

## 탐색

이동(위, 아래, 왼쪽, 오른쪽)을 위해 vim 탐색 키를 사용합니다.

- `j` → 아래로 이동
- `k` → 위로 이동
- `h` → 왼쪽으로 이동
- `l` → 오른쪽으로 이동

이동을 위해 화살표 키를 사용할 수도 있습니다:

- `↑` → 위로 이동
- `↓` → 아래로 이동
- `←` → 왼쪽으로 이동
- `→` → 오른쪽으로 이동

vim 키나 화살표 키 중 편한 것을 선택하면 됩니다.

---

## 스크립트 실행하기

스크립트 실행과 같은 빠른 작업을 위해:

- 오른쪽의 아무 스크립트로 이동한 다음 `Enter`를 눌러 실행합니다.
- Neovim 애호가라면 `l`을 눌러 실행할 수도 있습니다.

`y/n` 프롬프트가 나타나면:

- 승인(예)하고 실행하려면 `l`을 누릅니다.
- 이전 화면으로 돌아가려면 `h`를 누릅니다.

요약하면:

- `l`은 `Enter`와 동일하게 작동합니다
- `h`는 `Esc`와 동일하게 작동합니다

---

## 다중 선택 모드

여러 스크립트를 정확히 동시에 실행할 수는 없지만, 여러 스크립트를 선택하면 선택한 순서대로 하나씩 실행됩니다.

다중 선택을 사용하려면:

- 다중 선택 모드로 들어가려면 `m`을 누릅니다
- 스크립트를 선택하려면 `Space`를 누릅니다
- 실행하려면 `Enter`를 누릅니다

각 스크립트가 완료된 후 하나씩 실행됩니다.

---

## 테마

Carch TUI에는 여러 테마도 있습니다.

- 테마를 변경하려면 `t`를 누릅니다

좋아하는 테마로 Carch를 직접 실행하려면 문서에서 명령어를 확인할 수도 있습니다.

---

## 검색

왔다 갔다 하지 않고 스크립트를 빠르게 찾고 싶은 경우를 위해 검색 옵션도 있습니다.

- 검색 모드로 들어가려면 `/`를 누릅니다
- 스크립트 이름을 입력합니다
- 탭 자동 완성을 사용할 수 있습니다
- 해당 스크립트로 이동하려면 `Enter`를 누릅니다

---

## 미리보기

실행하기 전에 스크립트를 확인하려면:

- 스크립트를 미리 보려면 `p`를 누릅니다

이 기능은 실행하기 전에 스크립트가 무엇을 하는지 이해하는 데 도움이 됩니다.

---

## 설명

설명 모드는 스크립트에 대한 간단한 정보를 표시합니다. 이는 특히 초보자가 스크립트의 내용을 이해하는 데 유용합니다.

- 스크립트 목록이 있는 오른쪽으로 이동합니다
- 선택한 스크립트의 설명을 보려면 `d`를 누릅니다
