# 🖥️ CLI 명령어  

Carch는 CLI 명령어도 지원합니다! 다음 옵션과 함께 `carch`를 사용하세요:  

:::code-group

```sh [⚙️ CLI]
carch --help
```

:::

## 📦 사용법

carch [옵션] [명령어]

---

## 🔧 명령어 세부사항

| 🏷️ 명령어                  | 📄 설명                                                                                   |
|----------------------------|---------------------------------------------------------------------------------------------------|
| `setup`                    | 셸 완성, man 페이지 및 데스크톱 파일을 설치합니다.                                                   |
| `check-update`             | 새 버전이 있는지 확인합니다.                                                              |
| `completions <SHELL>`      | bash, zsh 또는 fish용 셸 완료를 생성합니다.                                                |
| `update`                   | Carch를 대화형으로 업데이트합니다.                                                                       |
| `uninstall`                | Carch를 대화형으로 제거합니다.                                                                    |
| `help`                     | 이 메시지나 주어진 하위 명령어에 대한 도움말을 출력합니다.                                           |
| `--version`, `-v`          | 프로그램 버전을 표시합니다.                                                                         |
| `--set-theme <THEME>`      | 기본 테마를 설정하고 저장합니다 (catppuccin-mocha, dracula, gruvbox, nord, rose-pine, 또는 default로 초기화)  |
| `--help`, `-h`             | 이 도움말 메시지를 표시하고 종료합니다.                                                                  |
